# Dataset_Distillation_for_NAS

# Repository Compilation for Dataset Distillation Research

This repository combines and adapts components from multiple open-source projects related to dataset condensation, dataset distillation, and neural architecture search.

The purpose of this repository is to provide a unified environment for experimentation, comparison, and development using methods derived from:

* Dataset Condensation
* TeslaNAS
* Matching Training Trajectories (MTT)

## Included Source Projects

This repository contains code adapted from the following repositories:

* Dataset Condensation

  * Repository: [https://github.com/VICO-UoE/DatasetCondensation](https://github.com/VICO-UoE/DatasetCondensation)
  * Authors: Bo Zhao, Konda Reddy Mopuri, Hakan Bilen

* TeslaNAS

  * Repository: [https://github.com/tsakman23/TeslaNAS](https://github.com/tsakman23/TeslaNAS)

* Matching Training Trajectories (MTT)

  * Repository: [https://github.com/GeorgeCazenavette/mtt-distillation](https://github.com/GeorgeCazenavette/mtt-distillation)
  * Authors: George Cazenavette et al.
